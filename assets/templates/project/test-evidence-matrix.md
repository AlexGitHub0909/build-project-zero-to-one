# Test evidence matrix

| Evidence ID | Requirement or risk | Priority | Environment | Type | Command or procedure | Expected result | Status | Evidence | Owner or blocker |
|---|---|---|---|---|---|---|---|---|---|

Allowed status values: `PASS`, `FAIL`, `PARTIAL`, `TODO / GAP`, `MANUAL_REQUIRED`, `BLOCKED`, `NOT_APPLICABLE`.

`TODO / GAP`, `PARTIAL`, and `MANUAL_REQUIRED` do not count as release passes.
